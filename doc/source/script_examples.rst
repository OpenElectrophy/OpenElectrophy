********************************
Simple examples for scripting
********************************


The best before learning to script with OpenElectrophy should be first to learn algorithm, then python language and at the end OpenElectrophy.
Since few neurobiologists have time to do this and because python is really an easy and intuitive language. The learning way can be done in reverse order!
First look at OpenElectrophy examples, then modify them and on the way improve your python knowledge.

Before starting reading, you have to be aware that it is easier to understand when the reader knows what a vector and a matrix are and how to manipulate them.
In short, if you ever programmed in matplab, R or python-scipy style, learning OpenElectrophy will take you few days (or hours).


The following list of examples are in the directory OpenElectrophy/examples. You can copy them before trying.
